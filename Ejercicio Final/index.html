Curso Programación Back-end

PHP es uno de los lenguajes mas utilizados debido a la variedad de clases y funciones disponibles.
Se considera un lenguaje de propósito general pero destinado especialmente al desarrollo de sitios Web.
Se encarga de programar scripts que se ejecutan del lado del servidor permitiendo generar páginas dinámicas.
Tiene acceso a una gran cantidad de base de datos (MySQL, PostgresSQL, SQLite, Oracle, entre otras) permite la generación dinámica de documentos (PDF, XLS, entre otros) y el uso de diferentes protocolos de internet (LDAP, IMAP, POP3, FTP, entre otros) Su sintaxis deriva del lenguaje C, claro que para desarrollar aplicaciones complejas hay que conocer sus particularidades.
